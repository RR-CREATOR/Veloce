# Veloce

3D-printer focusing on cheapness, speed, size and quality, for rapid prototyping and accessibility.


BOM: https://docs.google.com/spreadsheets/d/1xbmIo5lGiQjLDz0P9RclOuS1lZumxtib_qv9aH4qCCo/edit?usp=sharing

I want this printer to be fully upgradeable. I want it so as new and new innovations come out, it can be improved and made up to date, and thus also help save upgrade costs.
So made the printer around that.
